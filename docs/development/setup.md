# Development Setup

Follow these steps to set up your development environment for Wedy.

## Prerequisites

- Docker and Docker Compose
- Python 3.11+
- Flutter 3.x
- Git

## Quick Start

1. Clone the repository
2. Run the setup script: `./scripts/setup/dev-setup.sh`
3. Start development: `./scripts/dev/start-backend.sh`

For detailed instructions, see the main README.md file.
